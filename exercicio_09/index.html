Exercitando Funções

  1. Pedir nome da nave
  2. Velocidade inicial 0
  3. Menu deve ser exibido para que o usuário escolha:
    1. Acelerar a nave em 5km/s
    2. Desacelerar a nave em 5km/s (Por mais que desacelere, a nave não pode ter uma velocidade menor que 0km/s)
    3. Imprimir dados de bordo (Deve ser impresso o nome da nave com sua velocidade atual)
    4. Sair do programa
  1. Criar uma função que exiba o menu e valide a opção escolhida
  2. Uma função para cada ação do menu de acelerar, desacelerar e imprimir dados de bordo.
  3. Encerrar o app somente quando digitar para sair.
